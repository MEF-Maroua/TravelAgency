Step 1: Database Configuration
Open your MySQL administration interface (e.g., phpMyAdmin via XAMPP).
Create a new database named TravelAgency.
Import the TravelAgency.sql file located in the project's root directory into this database.

Step 2: Launch the Backend (The Server)
The backend manages data and users.
Open a terminal in the backend folder.
Install the dependencies
npm install
npm start

Step 3: Launch the Frontend (The Interface)
The frontend is the user interface you will see in the browser.
Open a second terminal in the frontend folder.
Install the dependencies
npm install
npm start
